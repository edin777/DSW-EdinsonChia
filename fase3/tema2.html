Las equietas HTML

El HTML se referencia como el lenguaje utilizado para el desarrollo de paginas Web, este a su vez se compone por diferentes tipos de etiquetas que al momento de cargar en el navegador, es interpretado por este, mostrando una apariencia que se ha establecido en su estructura de etiquetado

Una etiqueta de inicio esta presente en todo elemento de HTML, al igual que esta etiqueta de apertura debe contar con una etiqueta de cierre, estas dos etiquetas generan validez al elemento.
